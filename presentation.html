Project Overview & Objectives

Problem Statement

Educational institutions require an integrated, responsive ERP platform to track student admissions, academic CGPA analytics (10.0 scale), and departmental statistics without manual paperwork.

Core Objectives

  • Secure Administrator Authentication & Session Security.
  • Real-Time Academic Dashboard (Total, Avg/Max CGPA).
  • Full CRUD Operations (Create, Read, Update, Delete).
  • Multi-Keyword Live Search Filtering.
