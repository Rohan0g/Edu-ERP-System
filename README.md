## 🌟 Key Features

- **🔒 Login Screen**: Pre-filled credentials, session management using `sessionStorage`.
- **📊 Real-time Dashboard**: Dynamic KPI counters (Total Students, Avg GPA, Top GPA, Department share).
- **📁 Student Directory**: Full table with instant search filtering (Name, Roll ID, Email, Department).
- **➕ Add Student & ✏️ Edit Student**: Integrated form with real-time validation and local persistence.
- **🗑️ Delete Confirmation**: Interactive delete prompt with automatic UI update.
- **💾 LocalStorage Persistence**: Data stays saved even if you close or refresh your browser.
